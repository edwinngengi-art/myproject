# 🎵 Melodix | Pro Music Player

Melodix is a sleek, responsive music streaming interface built with modern web technologies. It allows users to search the iTunes library, preview 30-second tracks, and enjoy a high-fidelity dark-themed user experience inspired by Spotify.

**🚀 [Live Demo](https://edwinngengi-art.github.io/myproject/)**

---

## ✨ Features

- **Dynamic Search**: Real-time access to millions of songs using the iTunes Search API.
- **Custom Audio Engine**: Hand-built playback controls including play/pause, volume adjustment, and song scrubbing.
- **Responsive UI**: Styled with Tailwind CSS for a professional, mobile-friendly dark mode interface.
- **High-Res Visuals**: Automatically fetches and displays high-quality album artwork for the active track.
- **Real-time Feedback**: Dynamic progress bars and time stamps for a seamless listening experience.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for accessibility.
- **Tailwind CSS**: Utility-first CSS framework for modern styling.
- **JavaScript (ES6+)**: Custom logic for API fetching, DOM manipulation, and audio handling.
- **iTunes Search API**: External data source for music metadata and audio previews.
